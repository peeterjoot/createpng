== What is this ==

These are some simple standalone experiments to try to create a png file from C or C++.

== Dependencies ==

(fedora-33)

sudo dnf -y install libpng-devel
sudo dnf -y install ImageMagick
sudo dnf -y install ImageMagick-devel
sudo dnf -y install ImageMagick-c++
sudo dnf -y install ImageMagick-c++-devel
